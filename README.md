# Al-Wahidah

Qasidah pun memunculkan kesenian dengan alat musiknya yang paling utama adalah rebana, kecrek dan sebagaiya. Qasidah terdiri dari lima atau enam orang dalam memainkan berbagai ukuranalatnya dari yang kecil hingga rebana yang besar ditambah dengan alat kecrek. Pada perkembangan berikutnya qasidah dapat dimainkan menggunakan alat kesenian lainnya sesuai dengan keterampilan seniman itu sendiri.
